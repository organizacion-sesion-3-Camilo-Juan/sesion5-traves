# Ejemplo de proyecto de integración continua

Ejemplo de proyecto PHP con test unitarios para utilizar con Travis CI

## Instalación

``` bash
# Instalar dependencias
composer install
```

## Test

``` bash
vendor/phpunit/phpunit/phpunit tests/
```
